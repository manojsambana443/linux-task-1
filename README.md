# DevOps Internship – Task 1

## Task Name
Linux Environment Setup & Exploration

---

## Description
This task focuses on setting up a Linux environment and practicing basic terminal commands to understand how Linux systems work. The main goal is to get comfortable with navigation, file operations, permissions, and basic system monitoring.

---

## Tools Used
- Ubuntu (via WSL / VM)
- Linux Terminal
- GitHub

---

## What I Practiced
- Navigating directories using `pwd`, `ls`, and `cd`
- Creating and deleting files and folders
- Viewing and editing files using `cat`, `nano`, and `less`
- Understanding file permissions using `chmod`
- Monitoring system performance using `top`, `df`, and `free`
